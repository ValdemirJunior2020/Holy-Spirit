- **Church Partners Directory**

  - quick actions (call / email / maps)

  Visitors can submit reviews/testimonies and see latest reviews.

- **Visitor Tracking (Google Sheets DB)**  
  Basic pageview tracking (counts page views + unique-ish visitors using a local visitor id).
